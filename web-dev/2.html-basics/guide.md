# HTML Basics: Hello Web!

## Objective
Build your first webpage using HTML. You will learn the basic structure of an HTML document and how to add content like text, images, and links.

## Instructions

### Step 1: Create an HTML File
1.  Open VS Code.
2.  Create a new file named `index.html`.

### Step 2: HTML Structure
Learn the basic structure of an HTML document (tags like `<html>`, `<head>`, `<body>`).
*   **Read:** [W3Schools HTML Intro](https://www.w3schools.com/html/html_intro.asp)
*   Copy the example code from the tutorial into your `index.html` file.
*   **Run:** Open your file using Live Server. [How to enable Live Server](https://www.geeksforgeeks.org/how-to-enable-live-server-on-visual-studio-code/)

### Step 3: Add Content
Enhance your webpage by adding:
*   An image using the `<img>` tag.
*   A hyperlink using the `<a>` tag.
*   **Reference:** [HTML Attributes](https://www.w3schools.com/html/html_attributes.asp)
