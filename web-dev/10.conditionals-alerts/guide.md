# Using if...else Statements with Alerts

## Objective
Create an interactive script to check if a user input is positive, negative, or zero.

## Instructions

### Step 1: User Input
Prompt the user to enter a number using `prompt()`.

### Step 2: Logic
Use an `if...else` statement to check if the number is positive, negative, or zero.

### Step 3: Output
Display an alert message with the result.

**Example:**
```javascript
let number = prompt("Enter a number:");
if (number > 0) {
  alert("The number is positive.");
} else if (number < 0) {
  alert("The number is negative.");
} else {
  alert("The number is zero.");
}
```
