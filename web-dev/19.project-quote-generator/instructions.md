# Project: Dynamic Quote Generator

## Objective
Create a webpage that displays a new random quote every time a button is clicked.

## Instructions

### Step 1: API Setup
Use a random quote API (e.g., [Quotable API](https://api.quotable.io/random)).

### Step 2: Fetch Logic
Fetch a new quote when the user clicks the "New Quote" button.

### Step 3: Display
Display the quote and the author on the page.

### Step 4: Styling
Style the page to look appealing.
