# Error Handling and Loading States

## Objective
Add error handling and loading indicators to improve user experience.

## Instructions

### Step 1: Try...Catch
Add a `try...catch` block to handle errors (e.g., network issues).

### Step 2: Loading Indicator
Display a "Loading..." message while the data is being fetched and remove it once the data is ready.

### Step 3: Error Message
Display an error message if the fetch fails.

**Reference:** [Async/Await](https://javascript.info/async-await)
