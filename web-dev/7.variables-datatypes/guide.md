# Variables and Data Types

## Objective
Practice declaring variables and using different data types in JavaScript.

## Instructions

### Step 1: Declare Variables
Declare variables of each type (`string`, `number`, `boolean`, `array`, `object`).

**Example:**
```javascript
let name = "Amarnath Sujith";
let age = 34;
let isStudent = true;
let hobbies = ["reading", "coding"];
let address = { city: "Kazhakootam", country: "INDIA" };
```

### Step 2: Display Variables
Use `console.log()` to display each variable in the console.

**Reference:** [JavaScript Data Types](https://www.w3schools.com/js/js_datatypes.asp)
