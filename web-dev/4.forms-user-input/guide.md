# Forms and User Input

## Objective
Learn how to collect user input by building a contact form and a login page.

## Instructions

### Step 1: Build a Contact Form
Create a form with the following input fields:
*   Text (Name)
*   Email
*   Password
*   Submit Button

**Reference:** [HTML Forms](https://www.w3schools.com/html/html_forms.asp)

### Step 2: Style the Form
Add CSS to make your form look good.
**Reference:** [Style a Contact Form](https://www.w3schools.com/howto/howto_css_contact_form.asp)

### Step 3: Facebook Login Clone
Create a new file and try to clone the Facebook login page using HTML and CSS.
*   **Target Image:** [Facebook Login Clone](https://media.geeksforgeeks.org/wp-content/uploads/20240314120529/Screenshot-2024-03-14-at-12-04-38-GFG-FB-Clone.png)
*   **Challenge:** Try to do this by yourself first!
*   **Help:** Only if you are stuck, refer to this tutorial: [Create Facebook Login Page](https://www.geeksforgeeks.org/how-to-create-facebook-login-page-in-html-css/)
