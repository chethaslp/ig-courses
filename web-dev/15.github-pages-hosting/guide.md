# Host Your Project on GitHub Pages

## Objective
Publish your project on GitHub Pages to make it accessible online.

## Instructions

### Step 1: Create GitHub Repository
Create a new repository on GitHub and push your local project to it.

### Step 2: Enable GitHub Pages
1.  Go to your repository settings.
2.  Navigate to the "Pages" section.
3.  Select the branch (usually `main` or `master`) and save.

**Guide:** [How to Host a Website on GitHub](https://www.geeksforgeeks.org/how-to-host-a-website-on-github-for-free/)
