# Looping Through Arrays

## Objective
Create an array of your favorite movies and use a `for` loop to print each movie name.

## Instructions

### Step 1: Create Array
Declare an array with at least five movie names.
```javascript
let movies = ["Inception", "The Matrix", "Interstellar", "Shutter Island", "The Dark Knight"];
```

### Step 2: Loop
Use a `for` loop to iterate through the array and `console.log()` each movie name.
