# Project: Personal Linktree

## Objective
Create a fully styled, interactive Linktree-style website showcasing your social media handles.

## Instructions

### Task 1: Project Setup and HTML Structure
1.  **Create File:** Create an `index.html` file.
2.  **Structure:** Add the basic HTML structure (`<html>`, `<head>`, `<body>`).
3.  **Profile Section:** Add a `<div>` containing:
    *   An image `<img>` for your profile picture.
    *   A name and title (e.g., "Richard", "Developer").
4.  **Links Section:** Add buttons/links for LinkedIn, Instagram, Twitter, and Projects.
5.  **Reference:** [HTML Structure Example](https://github.com/richardshaju/socials/blob/master/index.html)

### Task 2: Styling with CSS
1.  **Create File:** Create a `style.css` file and link it to your HTML.
2.  **Layout:** Use Flexbox to center the container on the page.
    *   `display: flex`, `align-items: center`, `justify-content: center`.
3.  **Styling:**
    *   Make the profile picture circular (`border-radius: 50%`).
    *   Style the name and title (font size, color).
    *   Style the links to look like buttons (rounded rectangles).
4.  **References:**
    *   [CSS Styling](https://www.w3schools.com/css/)
    *   [CSS Flexbox](https://www.w3schools.com/css/css3_flexbox.asp)
    *   [CSS Example](https://github.com/richardshaju/socials/blob/master/style.css)

### Task 3: Final Polish
Customize the project to make it your own.
**Reference:** [Full Project Example](https://github.com/richardshaju/socials)
