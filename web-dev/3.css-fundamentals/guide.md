# CSS Fundamentals

## Objective
Make your webpage look better by learning basic CSS. CSS allows you to style your HTML elements.

## Instructions

### Step 1: CSS Methods
Learn how to apply CSS in three ways: Inline, Internal, and External.
*   **Read:** [Types of CSS](https://www.geeksforgeeks.org/types-of-css-cascading-style-sheet/)
*   Apply all three methods to style different elements in your HTML file.

### Step 2: CSS Properties
Use the following properties to style your elements:
*   `color`
*   `background-color`
*   `font-size`
*   `padding`
*   `margin`

**Reference:** [CSS Properties](https://www.geeksforgeeks.org/types-of-css-cascading-style-sheet/)

### Step 3: Experiment
Explore other CSS properties from the reference link and try adding them to your page.
