# Getting Started with JavaScript

## Objective
Write your first JavaScript program to display a message in the browser console.

## Instructions

### Step 1: Link JavaScript
Create a new HTML file and link an external JavaScript file.
**Reference:** [How to link JavaScript to HTML](https://www.scaler.com/topics/link-javascript-to-html/)

### Step 2: Console Log
In your JavaScript file, write the following code:
```javascript
console.log("Hello, JavaScript!");
```

### Step 3: Verify Output
Open your browser’s developer console to see the output (Press F12 or right-click > "Inspect" > Console).
