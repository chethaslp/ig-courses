# Mastering Arithmetic Operations

## Objective
Learn to perform arithmetic operations in JavaScript and display results.

## Instructions

### Step 1: Setup
Create an HTML file and link an external JavaScript file.

### Step 2: Declare Variables
Declare two variables, `num1` and `num2`, and assign them any numeric values.

### Step 3: Perform Operations
Perform the following operations on `num1` and `num2`:
*   Addition
*   Subtraction
*   Multiplication
*   Division
*   Modulus (remainder of division)

**Example Code:**
```javascript
let num1 = 10;
let num2 = 3;
console.log("Addition:", num1 + num2);
console.log("Subtraction:", num1 - num2);
console.log("Multiplication:", num1 * num2);
console.log("Division:", num1 / num2);
console.log("Modulus:", num1 % num2);
```

### Step 4: Run and Verify
Run your code and display each operation's result using `console.log()`.

**Reference:** [JavaScript Arithmetic](https://www.w3schools.com/js/js_arithmetic.asp)
