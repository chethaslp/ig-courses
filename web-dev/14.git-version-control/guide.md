## Exercise

### Step 1: Initialize Repository
Initialize a Git repository in your project folder.
```bash
git init
```

### Step 2: Commit Changes
Commit your initial HTML, CSS, and JavaScript files with a descriptive message.
```bash
git add .
git commit -m "Initial commit"
```

### Step 3: Update and Commit
Make a small update (e.g., change text or styles) and commit again.
