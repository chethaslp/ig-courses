# Responsive Web Design

## Objective
Ensure your webpage is responsive across devices. Responsive design allows websites to automatically adjust their layout to fit the screen size.

## Instructions

### Step 1: The Viewport Meta Tag
Learn about the meta viewport tag and add it to your HTML `<head>`.
**Reference:** [CSS Viewport](https://www.w3schools.com/css/css_rwd_viewport.asp)

### Step 2: Media Queries
Use media queries to adjust the layout for mobile devices.
**Reference:** [CSS Media Queries](https://www.geeksforgeeks.org/css-media-queries/)

### Step 3: Create a Responsive Page
Create a new web page that displays different styles on mobile and desktop.
**Example:** [Media Query Demo](https://media.geeksforgeeks.org/wp-content/uploads/20240725200510/mediaquery.mp4?_=2)
