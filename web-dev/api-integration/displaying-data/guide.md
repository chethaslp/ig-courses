# Displaying Data Dynamically

## Objective
Display the fetched data on your webpage.

## Instructions

### Step 1: Extract Data
Modify your JavaScript function to extract specific information (e.g., title, body) from the fetched data.

### Step 2: Update DOM
Use DOM manipulation to create HTML elements (like paragraphs or lists) and insert the data into the `div`.

**Reference:** [DOM Manipulation](https://www.w3schools.com/js/js_htmldom.asp)
