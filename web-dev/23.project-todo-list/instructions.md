# Project: To-Do List API

## Objective
Create a RESTful API for managing a to-do list.

## Instructions

### Step 1: Data Store
Create an in-memory array to store to-do items.

### Step 2: Routes
Implement GET, POST, PUT, and DELETE routes to manage the to-do list.

### Step 3: Test
Test all endpoints to ensure they work as expected.
